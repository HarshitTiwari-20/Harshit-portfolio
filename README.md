> A modern, highly interactive, and visually stunning personal portfolio built for a Full Stack Developer, DevOps Engineer, and Web3 Builder.

This repository contains the source code for my personal portfolio. It features a futuristic and dark "glassmorphism" aesthetic with smooth scroll animations, optimized assets, and a fully responsive mobile-first design.

---

## ✨ Key Features

- **Futuristic UI/UX:** Stunning fluid animations using [Framer Motion](https://www.framer.com/motion/) and an immersive dark-cyberpunk aesthetic.
- **Glassmorphism Design System:** Tailwind v4 driven customized UI focusing on frosted glass borders, glowing gradients, and glowing accents. 
- **Modular Components:** 
  - **Hero:** Captivating intro with animated gradients.
  - **About & Skills:** Highlighted tech-stack metrics organized via Frontend, Backend, DevOps, and Web3 capabilities.
  - **Experience:** Interactive vertical timeline showcasing past roles and impacts.
  - **Projects:** Dynamic grid highlighting decentralized products, dashboards, and automated CI pipelines.
  - **Contact:** Functional and fully validated form UI linked up with external social handles.
- **Modern Performance:** Optimized using Next.js 14 App Router, built-in SEO capabilities, and strict TS typings.

## 🛠️ Built With

- **[Next.js](https://nextjs.org/)** - React Framework providing App Router and Server Components.
- **[Tailwind CSS v4](https://tailwindcss.com/)** - Utility-first styling natively injected using `@theme`.
- **[Framer Motion](https://www.framer.com/motion/)** - The industry standard library for declarative UI animations.
- **[Lucide React](https://lucide.dev/)** - Clean and consistent SVG icons.
- **[TypeScript](https://www.typescriptlang.org/)** - For robust type-safety and developer tooling.

## 💻 Getting Started

Follow these steps to run the web application locally.

### Prerequisites

Ensure you have **Node.js** (v18+) and **npm** installed on your local machine.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/HarshitTiwari-20/Harshit-portfolio.git
   cd Harshit-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **View the application**
   Open your browser and navigate to [http://localhost:3000](http://localhost:3000).

## 🚀 Deployment

This project includes a fully configured **GitHub Actions** CI/CD workflow (`.github/workflows/deployment.yml`) that triggers on pushes and PRs to the `main` branch to ensure standard linting and build checks pass successfully.

To deploy the application securely for production on **Vercel** (Recommended for Next.js):
1. Connect this repository to your Vercel account.
2. The `deployment.yml` can be updated with your Vercel tokens to deploy programmatically, or you can allow the Vercel bot to auto-deploy natively on every push.
